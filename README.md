# Volvo 공식 웹사이트 메인 페이지 클론 프로젝트

## 프로젝트 한 줄 요약
글로벌 자동차 브랜드 **Volvo 공식 웹사이트 메인 페이지**를 HTML과 CSS로 클론 구현한 개인 프로젝트로,  
사용자 가독성을 최우선으로 고려한 **타이포그래피와 카드형 레이아웃 중심의 화면 구성**을 재현했습니다.

---

## 진행 기간
- 개인 프로젝트  
- 2025.10.24 ~ 2025.10.26 (2일)

---

## 주요 기능 & 구현 포인트
- 기업형 메인 페이지 레이아웃 구성  
- 카드 기반 콘텐츠 구조 설계  
- 브랜드 메시지 중심의 섹션 구성  
- 반응형 레이아웃 구현  
- 푸터 영역 정보 구조화  

---

## 기술 스택
- HTML5  
- SCSS  

---

## Trouble Shooting

### 문제
텍스트와 카드가 많은 페이지에서 정보가 한눈에 들어오지 않아  
전체적인 가독성이 떨어지는 문제가 발생했습니다.

### 해결
텍스트 구조를 다시 정리하여 **제목 / 본문 / 보조 정보**의  
글자 크기와 간격을 조정하고,  
카드 간 여백과 정렬을 통일해  
시각적으로 읽기 쉬운 레이아웃으로 개선했습니다.

---

## 개발 후 느낀 점
텍스트 양이 많고 카드가 반복되는 화면일수록  
단순히 디자인을 따라 구현하는 것이 아니라,  
**가독성과 정보 우선순위**를 기준으로 구조를 잡는 것이 중요하다는 점을 배웠습니다.  

이번 프로젝트를 통해 화면을 설계할 때  
사용자가 내용을 **쉽게 읽고 이해할 수 있는 흐름**을 먼저 고민해야 한다는  
기준을 갖게 되었습니다.
