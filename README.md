# Volvo Corporate Website Clone

## 한 줄 요약
글로벌 자동차 브랜드 Volvo 공식 웹사이트 메인 페이지를 HTML과 CSS 중심으로 구현한 클론 프로젝트로, 기업형 브랜드 사이트의 레이아웃 구성과 콘텐츠 흐름을 재현했습니다.

---

## 진행 기간
- 개인 프로젝트  
- 2025.10.24 ~ 2025.10.29 (5일)

---

## 주요 기능 & 구현 포인트

### 기업형 메인 페이지 레이아웃 구성
- 헤더, 메인 비주얼, 카드 섹션, 모델 소개, 브랜드 메시지, 푸터까지  
  실제 브랜드 사이트의 전체 흐름을 기준으로 화면 구성

### 카드 기반 콘텐츠 구조 설계
- 시승 신청, 전시장 찾기, 서비스 안내 등 반복되는 카드 UI를  
  섹션 단위로 정리해 정보 전달이 명확하도록 구성

### 브랜드 메시지 중심의 섹션 구성
- 전동화, 안전, 지속가능성 등 Volvo의 핵심 가치를 강조하는 콘텐츠 영역을  
  시각적으로 구분해 배치

### 반응형 레이아웃을 고려한 마크업
- PC / 태블릿 / 모바일 환경을 고려해 섹션과 요소 구조를 설계하고  
  CSS로 화면 크기별 레이아웃 대응 가능하도록 구현

### 푸터 정보 구조화
- 기업 사이트에서 자주 사용되는 고객지원, 앱 안내, 지역 선택, SNS 영역 등을  
  실제 서비스 구조를 참고해 구성

## Trouble Shooting

**문제:**  
섹션과 카드가 많은 기업형 페이지를 구현하면서 HTML 구조가 복잡해지고,  
일부 CSS 수정이 다른 영역의 레이아웃까지 영향을 주는 문제가 발생함

**해결:**  
반복되는 카드와 섹션을 기준으로 마크업 구조를 정리하고,  
공통 스타일과 영역별 스타일을 분리해 각 컴포넌트가 독립적으로 스타일링되도록 개선함
